Battle Royale Task Scheduler
A Clash Royale-themed smart task scheduling system built for Hackathon 2025.
What It Does
Transforms a list of tasks (with deadlines, durations, and difficulty ratings) into an optimized schedule that maximizes task completion while respecting deadlines and preventing cognitive overload from back-to-back difficult tasks.
DSA Concepts Used
•	Sorting (O(n log n)): Earliest Deadline First ordering
•	Min-Heap Priority Queue (O(log n)): Custom binary heap implementation
•	Greedy Algorithm: Optimal EDF scheduling policy
•	Difficulty Balancer: Post-processing pass for cognitive load management
